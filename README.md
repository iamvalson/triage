# Triage

##Project Description
Triage AI is a WhatsApp based AI assistant designed to provide instant, context-aware emergency health guidance and response. By leveraging the high accessibility of WhatsApp, the system offers users immediate first aid triage via conversational AI

##Installation Instructions:
```bash
git clone https://github.com/iamvalson/triage.git
cd triage
cd frontend
npm install
npm run dev
```

## Tech Stack
- TypeSCript
- React
- Tailwindcss
